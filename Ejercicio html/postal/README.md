Este proyecto consiste en una postal navideña interactiva construida bajo los conceptos de HTML y CSS, donde cada elemento se trata como una caja rectangular con propiedades de contenido, relleno y bordes. Utiliza un sistema de diseño bidimensional mediante CSS para organizar los elementos visuales de forma compleja y eficiente.
Características Principales
• Diseño Responsivo: La interfaz es adaptable a dispositivos móviles gracias al uso de media queries y un enfoque de diseño Mobile First.
• Estilos y Efectos: Implementa funciones avanzadas como clamp para tipografías flexibles y filter: DropShadow para añadir profundidad y realismo a las imágenes navideñas.
• Animaciones Dinámicas: Utiliza transiciones suaves y animaciones basadas en scroll para que los elementos aparezcan progresivamente mientras el usuario navega por la postal.
Instalación y Configuración (Git)
1. Clona el repositorio localmente: $ git clone [url].
2. Prepara los archivos para la versión: $ git add [archivo].
3. Registra los cambios en el historial: $ git commit -m "[mensaje descriptivo]".
4. Para visualizarla, abre el archivo index.html en un navegador web para que interprete el código HTML y CSS.
Reglas de Desarrollo
El código sigue las reglas del diseño simple y Clean Code, respetando el principio DRY (don't repeat yourself) para evitar la duplicación de lógica y revelando la intención del diseño mediante nombres descriptivos en variables y clases
